### 方法 3：重新打包（更新後）

當您修改程式碼後，重新打包：

```bash

powershell -ExecutionPolicy Bypass -File build-and-package.ps1

# 或手動
npm run build
Copy-Item QUICKSTART.md dist\
Compress-Archive -Path dist\* -DestinationPath DQA-SchedulePlanner.zip -Force
```
