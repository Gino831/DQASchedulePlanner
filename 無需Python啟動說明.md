## 💡 為什麼不需要 Python？

### 技術說明

此應用程式採用**現代化前端架構**：

```html
<!-- 使用 CDN 載入 React -->
<script type="importmap">
{
  "imports": {
    "react": "https://esm.sh/react@^19.2.3",
    "react-dom/": "https://esm.sh/react-dom@^19.2.3/"
  }
}
</script>

<!-- 使用 CDN 載入 Tailwind CSS -->
<script src="https://cdn.tailwindcss.com"></script>
```

**優點**：
- ✅ 所有資源從 CDN 載入
- ✅ 瀏覽器直接執行 JavaScript
- ✅ 無需編譯或建置環境
- ✅ 跨平台、跨瀏覽器

### 資料儲存

- 使用 **localStorage** 儲存資料
- 完全在瀏覽器本地運行
- 無需資料庫或後端伺服器

## 🌐 系統需求

**最低需求**：
- ✅ Windows 7 或更新版本
- ✅ 現代瀏覽器（Chrome、Edge、Firefox、Safari）
- ✅ 網路連線（僅首次載入 CDN 資源時需要）

**不需要**：

- ❌ 任何程式語言環境
