# DQA 簡易時程規畫 - 快速啟動指南

## 🚀 使用方式

### 方法 1：使用啟動器（最簡單！）推薦 ⭐

**Windows 使用者** - 雙擊以下任一檔案即可啟動：

1. **`啟動應用程式.bat`** - Windows 批次檔（適合一般使用者）
   - 雙擊即可在預設瀏覽器開啟
   - 無需安裝任何軟體

2. **`啟動應用程式.ps1`** - PowerShell 腳本（進階）
   - 功能與 .bat 相同，介面更美觀
   - 右鍵 → "使用 PowerShell 執行"

3. **`start.html`** - 美觀的啟動頁面
   - 雙擊開啟後點選「開始使用」按鈕
   - 提供功能介紹

> 💡 **最簡單的方式**：直接雙擊 `啟動應用程式.bat` 即可！

### 方法 2：直接開啟 HTML

1. 直接雙擊 `index.html` 檔案
2. 應用程式會在預設瀏覽器中開啟

> ⚠️ **注意**：此方法完全可用，所有功能都正常！應用程式使用 CDN 載入資源，無需本地伺服器。

### 方法 3：使用本地伺服器（進階，非必要）

如果您偏好使用本地伺服器：

#### 使用 Python（Windows/Mac/Linux 通用）

1. 開啟命令提示字元（Windows）或終端機（Mac/Linux）
2. 切換到此資料夾：

   ```bash
   cd [此資料夾的路徑]
   ```

3. 執行以下指令：

   ```bash
   # Python 3
   python -m http.server 8080
   ```

4. 在瀏覽器開啟：`http://localhost:8080`

#### 使用 Node.js（若已安裝）

```bash
npx serve
```

然後依照畫面指示在瀏覽器開啟對應網址。

## ❓ 常見問題

### Q1: 為什麼雙擊 index.html 後某些功能無法使用？

**A**: 現代瀏覽器對本地檔案有安全限制。建議使用本地伺服器運行（方法 2）。

### Q2: 我的資料會儲存在哪裡？

**A**: 資料儲存在瀏覽器的 localStorage 中，不會上傳到任何伺服器。清除瀏覽器資料時請小心。

### Q3: 可以在手機或平板上使用嗎？

**A**: 可以！但建議使用電腦以獲得最佳體驗。

### Q4: 如何備份我的設定？

**A**: 使用「匯出設定」功能，將設定儲存為 JSON 檔案。
